## Set up Shadcn

Shadcn UI is a collection of beautifully designed, accessible, and customizable React components that you can copy and paste into your apps. The setup process base on the [shadcn/ui](https://ui.shadcn.com) library, which provides a solid foundation for building modern user interfaces.

## Add a components

You can add components to your project using the `shadcn add` command.
